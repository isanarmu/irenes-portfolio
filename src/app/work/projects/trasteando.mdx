## What is Trasteando?

Trasteando is a full-stack storage marketplace that connects people looking for storage space with companies offering available units.

Users can search, compare, book and manage storage units. Companies can manage facilities, pricing, availability and client communication from their own dashboard.

---

## The product in 20 seconds

- **Marketplace:** clients can book storage units from companies.
- **Booking flow:** search, compare, reserve and pay.
- **Dashboards:** different experiences for clients, companies and admin users.
- **Real-time features:** chat between clients and storage providers.
- **Smart tools:** AI-assisted inventory categorization.
- **Stack:** React, Flask, PostgreSQL, SQLAlchemy, JWT, Stripe, Socket.IO, Google Maps and OpenAI.

---

## Main features

### Storage search and booking

Users can search for available storage units and start the booking flow directly from the platform.

This is the core user journey: finding a suitable storage space, checking the details and moving towards a reservation without having to contact several companies manually.

![Storage search screenshot](/images/projects/Trasteando/barra_busqueda.png)

### Location-based discovery

Google Maps helps users understand where each storage option is located and compare nearby alternatives visually.

This makes the marketplace more useful because location is one of the key decisions when choosing storage.

![Map screenshot](/images/projects/Trasteando/mapa.png)

### Secure login and role-based access

Trasteando uses JWT authentication and protected routes.

Different users access different areas depending on their role: client, company or admin.

![Login screenshot](/images/projects/Trasteando/trasteando_log_in.png)

### Company management

Companies can manage storage facilities, bookings, pricing, availability and client communication from their own dashboard.

This makes Trasteando a two-sided product, not just a landing page.

![Company management screenshot](/images/projects/Trasteando/company-management.png)

### AI-powered inventory

The inventory module uses AI to help users classify and organize stored items.

This adds value beyond booking: users can also manage what they store.

![inventator screenshot](/images/projects/Trasteando/ia.png)

## Technical build

Trasteando combines several real-world systems:

- **React** for the frontend interface.
- **Flask** for the backend API.
- **SQLAlchemy and PostgreSQL** for relational data modeling.
- **JWT** for authentication and protected routes.
- **Stripe** for payment flows.
- **Socket.IO** for real-time messaging.
- **Google Maps** for location intelligence.
- **OpenAI** for inventory categorization.
- **Railway, Render, Docker and Gunicorn** for deployment preparation.
