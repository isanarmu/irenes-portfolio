## What is Trasteando?

Trasteando is a storage marketplace.

Users can find storage spaces, compare options, book a unit and manage their items. Storage companies can manage facilities, prices, availability, messages and client activity from their own dashboard.

---

## The project in 20 seconds

- **Type:** full-stack marketplace
- **Users:** clients, companies and admin roles
- **Main flow:** search → compare → book → pay → manage storage
- **Core features:** maps, payments, chat, dashboards and AI inventory
- **Stack:** React, Flask, PostgreSQL, SQLAlchemy, JWT, Stripe, Socket.IO, Google Maps and OpenAI

---

## What I wanted to solve

Finding storage space should not feel like calling ten companies and comparing prices manually.

The idea was to build a product where users can see available storage options, understand the location, book securely and communicate directly with the provider.

For companies, the goal was to create a management area where they could control their storage units, bookings, pricing and client communication.

---

## Main features

### 1. Storage search and booking

Users can browse available storage units and start a booking flow from the platform.

This part connects the user interface with backend availability, booking data and payment preparation.

![Book a storage screenshot](/images/projects/trasteando/trasteando_book_a_storage.png)

### 2. Location-based discovery

Google Maps helps users understand where each storage option is located and compare nearby alternatives visually.

![Map screenshot](/images/projects/trasteando/mapa.png)

### 3. Authentication and roles

The platform uses JWT authentication and protected routes.

Different users see different areas depending on their role: client, company or admin.

![Login screenshot](/images/projects/trasteando/trasteando_log_in.png)

### 4. Company management

Companies can manage facilities, availability, pricing and conversations from a dedicated dashboard.

The project was designed as a two-sided platform, not just a public website.

### 5. AI inventory tools

The inventory module uses AI to help classify and organize stored items.

This makes the product more useful than a basic booking platform, because it also helps users manage what they store.

## Technical build

The project combines several real-world systems:

- **React** for the frontend interface.
- **Flask** for the backend API.
- **SQLAlchemy and PostgreSQL** for relational data modeling.
- **JWT** for authentication and protected routes.
- **Stripe** for secure payment flows.
- **Socket.IO** for real-time communication.
- **Google Maps** for location intelligence.
- **OpenAI** for AI-powered inventory categorization.
- **Railway, Render, Docker and Gunicorn** for deployment preparation.
